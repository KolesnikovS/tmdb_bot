# tmdb_bot
A simple telegram bot to get movie data from tmdb and form a link to the video
It allows to find movies in English while searchimg in Russian
##Connect to TMDb
You need a tmdb token to connect to the api. You can get one here:
https://developers.themoviedb.org/4/auth/create-access-token
##Connect to streaming
You can change the base_movie_url parameter according to the video streaming service you use. My service uses a tmdb id, which allows me to get a direct link to the movie.
